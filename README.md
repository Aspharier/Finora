<h1 align="center">Finora</h1>

<p align="center">
  A sleek & intuitive Personal Finance and Expense Tracker app built for Android using Kotlin & Jetpack Compose
</p>

---

**Finora** is a feature-rich **personal finance and expense tracking app** built with **Kotlin** and **Jetpack Compose**.  
It helps you **track income, expenses, budgets, and savings** with a clean UI and powerful analytics

Think of it as your **digital finance companion** to stay on top of your money.

## Features

- Add and track **expenses** and **income**
- View **daily / weekly / monthly spending**
- Categorize transactions  
- Built-in **analytics & charts**
- Manage budgets and savings goals  
- Clean & modern UI using **Jetpack Compose**
- Lightweight and responsive

---

## Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Room Database**
- **Hilt** (Dependency Injection)
- **Coroutines**
- **MVVM Architecture**
- Optionally **Charts Library** (for graphs)
